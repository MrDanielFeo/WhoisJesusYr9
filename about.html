Why I chose a web design

1. I chose to construct a website because coding websites and creating them interests me and its a way for me to learn new coding techniques. This serves for two purposes and that is to present an assignment as well as learning to code better.


2. Using a website lets me express my ideas more clearly because it is in a field that makes it fun for me.

3. I chose a website because it is more interesting than paragraphs on a page or a magazine type presentation which is presented on a flat stye presentation. With a website is fun and interactive for the viewer and the creator.
